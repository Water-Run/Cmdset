# Cmdset  

一些在Windows使用的命令集.  
安装: 将此目录拉下后添加至环境变量, 并放在前头.  

| 命令       | 说明                                                                            |
|------------|---------------------------------------------------------------------------------|
| `reboot`   | 重启                                                                            |
| `shutdown` | 额外增加`shutdown now`                                                          |
| `pmc`      | 代码上下文打包工具, 见[pack-my-code](https://github.com/Water-Run/pack-my-code) |
| `treepp`   | 更好的Windows tree命令, 见[treepp](https://github.com/Water-Run/treepp)         |
